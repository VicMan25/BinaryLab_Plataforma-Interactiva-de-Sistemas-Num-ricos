Sistema de Numeración Binaria

Descubre cómo las computadoras entienden el mundo con solo dos dígitos: 0 y 1.

01

¿Qué es el sistema binario?

El sistema binario es un sistema de numeración de base 2. A diferencia del sistema decimal (base 10) que usa los dígitos 0-9, el binario solo usa 0 y 1.

Estos dos estados representan perfectamente los dos estados eléctricos de un circuito: apagado (0) y encendido (1).

💡 Toda información en una computadora —texto, imágenes, videos— se almacena como secuencias de 0s y 1s.

¿Cómo se convierte?

Multiplica cada bit por su potencia de 2 correspondiente y suma todos los resultados:

1 0 1 1
= 1×2³ + 0×2² + 1×2¹ + 1×2⁰
= 8 + 0 + 2 + 1
= 11

→B

Método de Divisiones por 2

Divide el número entre 2 repetidamente. Los residuos, leídos de abajo hacia arriba, forman el número binario.

13 ÷ 2= 6R: 1
6 ÷ 2= 3R: 0
3 ÷ 2= 1R: 1
1 ÷ 2= 0R: 1
13₁₀ = 1101₂ (leer ↑)

¿Qué es una dirección IPv4?

Una dirección IPv4 es un identificador único de 32 bits que se asigna a cada dispositivo en una red.

Se escribe como 4 octetos (grupos de 8 bits) separados por puntos. Cada octeto puede tener un valor del 0 al 255.
